# Hitech HRX - Advanced HR Management System

![Hitech HRX](https://raw.githubusercontent.com/csenerds-web/hrx/main/public/logo.png)

Hitech HRX is a state-of-the-art Human Resources Management System designed to streamline organizational workflows, manage employee performance, and automate payroll processes. Built with performance and user experience in mind, HRX provides a premium interface for both administrators and employees.

## 🚀 Features

- **Employee Portal**: Comprehensive dashboard for personal information, leave management, and performance tracking.
- **Attendance Tracking**: Sophisticated attendance monitoring with manual credit options.
- **Payroll Automation**: Automated salary calculations and payroll adjustments.
- **Performance Management**: KPI/KRA based performance tracking with grading systems.
- **Notification System**: Real-time updates for key events like password changes and leave approvals.

## 🛠️ Technology Stack

- **Backend**: Laravel 10/11 (PHP)
- **Frontend**: Blade Templates, SCSS, JavaScript (Vanilla & Modern Libraries)
- **Build Tool**: Vite
- **Database**: MySQL / PostgreSQL
- **Security**: Robust authentication gates and security auditing tools.
